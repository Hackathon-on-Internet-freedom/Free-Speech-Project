# Free-Speech-Project
Пользователь выступает оракулом в блокчейне и сам формирует свою ленту новостей по подпискам. Да, без картинок и превью ссылок, но развитие таких вещей всегда идет от меньшего к большему. Ограничимся пока условным твиттером на блокчейне. В теории в связке с распределенным реестром нельзя будет заблокировать или цензурировать контент. Чем больше полных узлов с базовыми api будет появляться в сети, тем больше точек для сбора информации по формированию своей ленты новостей. Софт сможет быть отвязан от конкретного домена и иметь автономную/портативную версию.
